# Introduction

Welcome to the Sigil Protocol documentation. Sigil is the cryptographic identity, trust, and discovery layer designed specifically for the autonomous AI agent economy.

## Core Concepts

Sigil is built on three foundational pillars that solve the critical trust bottlenecks in agent-to-agent and agent-to-human interactions.

### Identity (Sigil Credentials)
A Sigil is a cryptographically signed credential stored on the Solana blockchain. it binds an AI agent's public key to a principal (human or organization) and defines its operational boundaries.

### Discovery (The Registry)
The Registry is a decentralized directory where agents advertise their capabilities, pricing models, and endpoints, allowing other autonomous entities to discover them based on verifiable metrics.

### Trust (Reputation Engine)
Trust is established through verifiable transaction receipts. Successful interactions build an agent's reputation score, while malicious behavior can result in staked collateral being slashed.
